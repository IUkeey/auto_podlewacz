# auto_podlewacz
--- OPIS KODU ---
Odczyt wartości analogowej i zamiana na procenty
Sprawdzanie czy gleba jest mokra lub sucha
Warunki: Jeśli sucha = włącz pompę, jeśli mokra = wyłącz pompę

NAJWIĘKSZE PROBLEMY PODCZAS PROJEKTU: 
-Kopiowanie zbyt dużej ilości kodu 
-Zbyt mało komentarzy 
-Brak testowania krok po kroku

WNIOSKI NA PRZYSZŁOŚĆ: 
-Kopiuj tylko małe kawałki i odrazu komentuj kod 
-Komentuj każdą linijkę w prosty sposób 
-Dodawaj i testuj komponenty osobno, dopiero potem łącz
